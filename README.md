# Smart Study Note Converter 🧠

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![NLTK](https://img.shields.io/badge/Library-NLTK-green.svg)
![Status](https://img.shields.io/badge/Status-Completed-success.svg)

A Python automation tool that uses **Natural Language Processing (NLP)** to read raw text notes, parse sentence structures, identify key concepts, and automatically generate fill-in-the-blank flashcards. The generated flashcards are exported directly to a CSV file, making them ready to import into study apps like Anki or Quizlet.

---

## 🚀 Features

* **Automated Parsing:** Reads through bulk text files seamlessly.
* **Smart Concept Identification:** Uses the `NLTK` library to grammatically tag words and automatically isolate nouns (key concepts).
* **Dynamic Generation:** Replaces chosen concepts with blanks to create challenging "fill-in-the-blank" style questions.
* **CSV Export:** Automatically formats and saves the generated questions and answers into a `flashcards.csv` spreadsheet.
* **Anki/Quizlet Ready:** The output CSV format is universally compatible with the most popular flashcard software.

---

## 🛠️ Prerequisites

To run this project, you will need Python installed on your machine along with the Natural Language Toolkit (`nltk`) library.

```bash
pip install nltk
```

---

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/zainabid67/Flashcard-generator.git
   cd Flashcard-generator
   ```

2. **Add your study materials:**
   Open the `notes.txt` file and paste your raw study notes into it. Make sure you use proper punctuation (periods at the end of sentences) so the program knows where sentences begin and end.

3. **Run the script:**
   ```bash
   python converter.py
   ```

*(Note: The very first time you run the script, it may take a few extra seconds to download the necessary NLTK grammar rules.)*

---

## 📖 Usage Example

**1. Input (`notes.txt`):**
> Kinetic energy is the energy that an object possesses due to its motion.
> A CPU is the primary component of a computer that acts as its brain.

**2. Execution:**
Run `converter.py`. The script will process the text and isolate the nouns.

**3. Output (`flashcards.csv`):**
| Question | Answer |
| :--- | :--- |
| Kinetic energy is the energy that an object possesses due to its ________. | motion |
| A CPU is the primary component of a ________ that acts as its brain. | computer |

---

## 📂 Project Structure

* `converter.py` - The main Python script containing the NLP and CSV export logic.
* `notes.txt` - The input text file where you place your raw study material.
* `flashcards.csv` - The output file generated by the script (appears after running).
